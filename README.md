## 📌 Sobre o Projeto
Este projeto consiste na reestruturação de um relatório financeiro criativo, focando na **Experiência do Usuário (UX)**. O objetivo foi elevar a maturidade visual e funcional do dashboard, aplicando princípios de design para facilitar a tomada de decisão.

O desafio faz parte da trilha de aprendizado voltada para a criação de relatórios de alto impacto, onde a estética e a usabilidade caminham juntas com a análise de dados.

---

## 📖 Ementa e Etapas de Desenvolvimento
O desenvolvimento foi dividido em 7 pilares fundamentais:
1. **Entendendo o desafio:** Alinhamento de objetivos.
2. **Adequando Relatório – Parte 1:** Início da reestruturação visual.
3. **Adequando Relatório – Parte 2:** Refinamento de elementos e lógica.
4. **Considerações sobre visuais:** Modificação técnica do gráfico de área para melhor leitura.
5. **Matriz de Descrição de Vendas:** Criação de detalhamento granular.
6. **Estilização:** Ajuste fino de cores, fontes e backgrounds.
7. **Navegabilidade:** Implementação de botões e fluxo de páginas.

---

## 🎨 Foco na Experiência do Usuário (UX)
Para este projeto, foram aplicados conceitos que transcendem a simples exibição de dados:

*   **Posicionamento:** Organização estratégica seguindo o fluxo de leitura ocidental (Z-Pattern).
*   **Contraste:** Diferenciação clara entre elementos de fundo e KPIs críticos.
*   **Proporção Áurea:** Utilizada para definir a hierarquia de tamanhos entre os visuais.
*   **Segmentação de Dados:** Filtros intuitivos para permitir que o usuário personalize sua visão rapidamente.
*   **Criatividade:** Quebra de regras rígidas em pontos específicos para tornar o relatório mais atraente e único.
